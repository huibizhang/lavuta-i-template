# Lavuta + i template

- Laravel 8
- Vue 3
- Tailwind CSS 2.2
- Inertia.js

### After Download, Before Run:

1. run `composer install` to get ./vender back.
2. run `npm install` to install node modules. 
3. change `.env.example` file name to `.env`
4. run `php artisan key:generate` to generate key in **.env**
